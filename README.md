# Washop: High-Performance WhatsApp Commerce Framework

**Washop** is a professional-grade WordPress framework engineered for businesses that need to bypass traditional checkout friction and close sales directly via WhatsApp. 

Developed by **Kaleemullah Naik (Onlinetist)**, this project open-sources the custom child-theme logic required to transform a standard WooCommerce installation into a high-converting, mobile-first business asset.

## 🚀 Technical Highlights
- **Custom-Coded Redirect Logic:** Hand-crafted PHP hooks that intercept the WooCommerce checkout process and generate structured WhatsApp order data.
- **Ultra-Lightweight Architecture:** Built as an optimized Child Theme for Astra, ensuring zero bloat and elite **Core Web Vitals** performance.
- **PWA Integration:** Native support for Progressive Web App (PWA) functionality, making your website installable on iOS and Android.
- **Conversion-Optimized UI:** Designed specifically for the high-intent, mobile-first market in Dubai and the UAE.

## 🛠 Installation & Deployment
This framework is delivered as a lightweight Child Theme containing all the custom functions and styles.

1. **Environment Setup:** Install a clean version of **WordPress**.
2. **Core Dependencies:** Install and activate the **Astra Theme** and the **WooCommerce** plugin.
3. **Theme Deployment:** 
   - Access your server via File Manager or FTP.
   - Navigate to `/public_html/wp-content/themes/`.
   - Upload and extract the `washop-child.zip` file.
4. **Activation:** Go to your WordPress Dashboard > Appearance > Themes and activate the **Washop Child Theme**.
5. **Final Touch:** Your high-performance WhatsApp shop is now active and ready for product configuration.

## 📦 What’s Inside the Theme?
- `functions.php`: Custom logic for cart-to-WhatsApp data parsing.
- `style.css`: Performance-tuned CSS for mobile-first user experiences.
- `pwa-logic/`: Manifest and configuration for "Add to Home Screen" support.
